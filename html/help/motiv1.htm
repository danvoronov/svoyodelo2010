Мотивация «Воздействие, сила»

Потребность во власти типична для людей, которые предпочитают держать все под контролем. Ориентация на самоуверенность.

Некоторые характеристики:

  • наслаждаются конкуренцией и положением, ориентированным на статус;
  • стараются найти слабые места, с помощью которые можно захватить управление;
  • в основном занимают позиции связанные с непосредственным руководством людьми.

Для людей власти основными ценностями являются:

  • влияние (способность изменять поведение других людей);
  • воздействие (способность эмоционально вовлекать);
  • контроль над другими;
  • престиж, статус, уважение других людей.
